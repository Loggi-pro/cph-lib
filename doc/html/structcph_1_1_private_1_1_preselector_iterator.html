template<uint32_t BASE_VALUE, bool IsTryNext, uint8_t N>
struct cph::Private::PreselectorIterator< BASE_VALUE, IsTryNext, N >

Declare iterator throught an array of PrescalarSelector.

Variant 3: Prescalar is still bigger,then return Stopped and catch it for OVERFLOW.

Variant 2: Current Prescalar is bigger than BASE. Then return value of this prescalar.

First Step: Check that this next element is lower that BASE. Based on this condition selected one of three variants Variant 1. Repeat this step.
